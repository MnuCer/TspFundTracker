# TSP Fund Tracker — Design Ideas

## Approach A — "Terminal Finance" Dark Monochrome
<response>
<text>

**Core Principles:** Dense data density, monospace type, high-contrast dark background, data-first hierarchy
**Color Philosophy:** Near-black (#0a0a0f) base with electric green (#00ff88) for gains, red (#ff3366) for losses, amber (#ffaa00) for neutral/G-fund. Inspired by trading terminals.

**Animation:** Minimal — only number transitions and chart line draws
**Typography:** JetBrains Mono for numbers/data, IBM Plex Sans for labels

## Approach B — "Precision Dashboard" Dark Navy Professional
<response>
<text>
**Design Movement:** Modern Financial Dashboard — Bloomberg meets Linear.app
**Core Principles:** Structured grid, sharp contrast, data clarity, subtle depth through layering
**Color Philosophy:** Deep navy (#0d1117) background, electric blue (#2563eb) primary, emerald (#10b981) for gains, rose (#f43f5e) for losses. Professional yet modern.
**Layout Paradigm:** Left sidebar navigation with collapsible panels, main content area with asymmetric grid — ticker strip at top, charts dominate center, analytics panels on right
**Signature Elements:** Glassmorphism cards with subtle borders, gradient chart fills, animated number counters
**Interaction Philosophy:** Smooth transitions, hover states with glow effects, interactive chart crosshairs
**Animation:** Entrance animations for cards, smooth chart draws, number counting animations
**Typography:** Space Grotesk for headings (geometric, modern), DM Mono for numbers
</text>

**Animation:** Scan-in animations, data loading sequences, smooth state transitions

## Selected Approach: **B — "Precision Dashboard" Dark Navy Professional**

This approach best balances professional finance aesthetics with modern UI clarity. The deep navy + electric blue palette conveys trust and precision, while the clean grid layout maximizes data density without feeling overwhelming. The glassmorphism cards and gradient charts give it a premium feel appropriate for a federal retirement tracking tool.
